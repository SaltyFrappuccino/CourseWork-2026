# Дробные производные Грюнвальда-Летникова

Программный комплекс к курсовой работе "Таблица дробных производных
Грюнвальда-Летникова элементарных функций". Считает дробные производные
элементарных функций по формуле Грюнвальда-Летникова, строит таблицы и
графики, проверяет точность на целых порядках.

## Что внутри

| Файл | Назначение |
| --- | --- |
| `fractional.py` | коэффициенты `c_k` и дробная производная в точке / на сетке |
| `functions.py` | набор функций (sin, cos, exp, степенные, ln, ...) с эталонами |
| `experiments.py` | расчётные эксперименты, возвращают таблицы чисел |
| `tables.py` | экспорт таблиц в CSV и `tables.xlsx` |
| `plots.py` | построение графиков в `graphs/` |
| `main.py` | точка входа: расчёт из командной строки и сборка всего |
| `web/` | отдельный сайт на React/Vite/Plotly для статического деплоя |
| `tests/` | автоматические тесты реализации (`pytest`) |
| `report/coursework.tex` | текст курсовой (LaTeX -> `coursework.pdf`) |

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

Дробная производная в одной точке:

```bash
python main.py --function sin --alpha 0.5 --h 0.01 --n 500 --x 1.5708
```

Таблица на сетке с сохранением в CSV:

```bash
python main.py --function x2 --alpha 0.5 --h 0.01 --n 500 \
               --xmin 1 --xmax 5 --points 9 --csv out.csv
```

Собрать все таблицы (`tables/`) и графики (`graphs/`) работы:

```bash
python main.py --all
```

Веб-интерфейс на React с интерактивным графиком:

```bash
cd web
bun install
bun run dev
```

Сборка сайта для деплоя:

```bash
cd web
bun run build
```

Автоматические тесты реализации:

```bash
pytest
```

## Метод кратко

Левая дробная производная порядка a считается как

```
D^a f(x) ~= h^(-a) * Sum_{k=0..N} c_k * f(x - k*h),   c_k = (-1)^k * C(a, k)
```

Коэффициенты - по рекуррентной формуле `c_0 = 1`, `c_k = c_{k-1}*(k-1-a)/k`.
Метод имеет первый порядок точности O(h); для дробного a важна длина истории
`L = N*h`, потому что коэффициенты затухают медленно, как `|c_k| ~ k^(-(a+1))`.
